# Функции

## Что такое функция

Функция — это блок кода, который можно вызывать несколько раз.

---

## Зачем нужны функции

- не дублировать код  
- сделать код понятнее  
- разбить задачу на части  

---

## Как создать функцию

```python
def hello():
    print("Привет")
```

Вызов:

```python
hello()
```

Результат:
```
Привет
```

---

## Параметры

```python
def greet(name):
    print("Привет,", name)

greet("Иван")
```

Результат:
```
Привет, Иван
```

---

## Несколько параметров

```python
def add(a, b):
    print(a + b)

add(2, 3)
```

Результат:
```
5
```

---

## return

Возвращает результат из функции.

```python
def add(a, b):
    return a + b

result = add(2, 3)
print(result)
```

Результат:
```
5
```

---

## Разница между print и return

```python
def test():
    print(5)

def test2():
    return 5

test()
print(test2())
```

Результат:
```
5
5
```

---

## Значения по умолчанию

```python
def greet(name="Гость"):
    print("Привет,", name)

greet()
greet("Иван")
```

Результат:
```
Привет, Гость
Привет, Иван
```

---

## Локальные переменные

Переменные внутри функции существуют только внутри неё.

```python
def test():
    x = 10
    print(x)

test()
```

Результат:
```
10
```

Попытка обратиться снаружи:

```python
print(x)
```

Результат:
```
Ошибка
```

---

## Простой пример

```python
def is_even(number):
    return number % 2 == 0

print(is_even(4))
print(is_even(5))
```

Результат:
```
True
False
```

---

## Частые ошибки

❌ Забывают return

```python
def add(a, b):
    a + b   # ничего не вернет
```

❌ Путают print и return  

❌ Неправильное количество аргументов

```python
def add(a, b):
    return a + b

add(1)  # ошибка
```

---

## Что запомнить

- def — объявление функции  
- параметры — входные данные  
- return — результат  
- print — просто вывод  
- функции делают код чище
