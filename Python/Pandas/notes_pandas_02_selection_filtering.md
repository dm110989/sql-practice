# Pandas: выборка и фильтрация данных

## Выбор столбца

```python
print(df["name"])
```

Результат:
```
0    Иван
1    Петр
Name: name, dtype: object
```

---

## Выбор нескольких столбцов

```python
print(df[["name", "age"]])
```

Результат:
```
   name  age
0  Иван   25
1  Петр   30
```

---

## Фильтрация по условию

```python
print(df[df["age"] > 25])
```

Результат:
```
   name  age
1  Петр   30
```

---

## Несколько условий

```python
print(df[(df["age"] > 20) & (df["age"] < 30)])
```

Результат:
```
   name  age
0  Иван   25
```

---

## Важно

- Используем `&` вместо `and`
- Используем `|` вместо `or`
- Каждое условие в скобках

---

## loc

Выбор по названию строк и столбцов

```python
print(df.loc[0, "name"])
```

Результат:
```
Иван
```

---

## iloc

Выбор по индексам

```python
print(df.iloc[0, 1])
```

Результат:
```
25
```

---

## Срезы

```python
print(df.loc[0:1])
```

Результат:
```
   name  age
0  Иван   25
1  Петр   30
```

---

## isin()

Проверка вхождения

```python
print(df[df["name"].isin(["Иван"])])
```

Результат:
```
   name  age
0  Иван   25
```

---

## Добавление условия в переменную

```python
condition = df["age"] > 25
print(df[condition])
```

Результат:
```
   name  age
1  Петр   30
```

---

## Частые ошибки

❌ Используют and/or вместо &/|

```python
df[df["age"] > 20 and df["age"] < 30]  # ошибка
```

❌ Забывают скобки

```python
df[df["age"] > 20 & df["age"] < 30]  # ошибка
```

---

## Что запомнить

- df["col"] — выбор столбца  
- df[["col1", "col2"]] — несколько столбцов  
- df[условие] — фильтрация  
- & и | — для нескольких условий  
- loc — по названиям  
- iloc — по индексам  
- isin() — проверка значений
